
# 🏠 House Prices Prediction using XGBoost & GPU
-----------------------------------------------------------------------------------------------------------------------

هذا المشروع يهدف إلى توقع أسعار المنازل في مدينة Ames، ولاية آيوا، باستخدام خوارزمية XGBoost المتقدمة مع الاستفادة من قوة المعالجة في كرت الشاشة (GPU).

## 🚀 ملخص النتائج
* الدقة (R-squared): 90.75%
* متوسط الخطأ (RMSE): $24,736
* المعالج المستخدم: NVIDIA GeForce RTX 2050 (عبر تقنية CUDA)

## 🛠️ الأدوات والتقنيات المستخدمة
- Python 3.x
- Pandas & NumPy: لمعالجة وتجهيز البيانات.
- Scikit-learn: لتقسيم البيانات وتقييم الأداء.
- XGBoost: لبناء نموذج التوقع (Regression Model).
- Matplotlib & Seaborn: لتصوير البيانات وتحليل أهم الميزات.

## 🧠 خطوات العمل (Workflow)

### 1. تنظيف البيانات (Data Cleaning)
- .
- معالجة القيم المفقودة (NaN) باستخدام التوزيع النسبي للقيم الأصلية لضمان عدم حدوث انحياز (Bias).
- التعامل مع الأعمدة النصية وتحويلها إلى أرقام باستخدام One-Hot Encoding مع تجنب "فخ المتغيرات الوهمية" (Dummy Variable Trap).


### 2. تدريب النموذج (Model Training)
تم استخدام إعدادات احترافية لضمان أفضل أداء:
- n_estimators=1000
- learning_rate=0.05
- max_depth=5
- استخدام tree_method='hist' مع device='cuda' لتسريع التدريب 10 أضعاف باستخدام الـ GPU.

## 📊 أهم الميزات المؤثرة (Feature Importance)
أظهر النموذج أن العوامل التالية هي الأكثر تأثيراً على سعر المنزل:
1. المساحة الكلية للمعيشة (GrLivArea).
2. جودة التشطيب العام (OverallQual).
